# mine sweeper
## mine sweeper is cloned from classic windows game, written with PyQt

### Dependency:
* python >= 3.7
* PyQt5 (python and Qt5 bindings)
* numpy (optional)

### Run:
* navigate to source dir in console
* type: `python ms.py` in console to run the application

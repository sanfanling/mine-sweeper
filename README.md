# mine-sweeper
## mine-sweeper is a linux clone of windows classic game, written by PyQt

### Dependency:
* python >= 3.6
* PyQt5 (python and Qt5 bindings)
* numpy (optional)

### Run:
* navigate to source dir in console
* type: `python ms.py` in console to run the application
